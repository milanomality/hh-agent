# HH-Agent

Личный ИИ-агент поиска работы на hh.ru: мониторит новые вакансии, оценивает их против
вашего резюме (Claude), пишет сопроводительные письма и присылает карточки в Telegram.
Отклик совершаете вы сами по кнопке-ссылке — с 15.12.2025 соискательский API hh закрыт,
автоотклики невозможны (и не эмулируются: это нарушение правил площадки).

Архитектура и план — в [PLAN.md](PLAN.md).

## Быстрый старт

1. Python 3.11+, зависимости:
   ```
   python -m venv .venv
   .venv\Scripts\python -m pip install -e .[dev]
   ```
2. Скопируйте `.env.example` → `.env` и заполните:
   - Telegram-бот — @BotFather (токен), свой chat_id — @userinfobot;
   - LLM_API_KEY — бесплатный ключ Groq (https://console.groq.com); альтернативы
     (OpenRouter / Gemini / локальная Ollama) — см. комментарии в `.env.example`.
     Платные модели по умолчанию не используются.
3. Положите резюме в `resume.md` (обычный текст или markdown: опыт, навыки, ожидания).
4. Запуск из корня проекта:
   ```
   .venv\Scripts\python -m hh_agent.main
   ```
5. В Telegram: `/add python разработчик` — добавить поиск. Агент присылает карточки
   подходящих вакансий с оценкой и готовым письмом; откликаетесь по кнопке
   «Открыть на hh.ru», затем жмёте «✅ Откликнулся» — факт попадает в воронку (`/status`).

Примечание: первый проход нового поиска может оценить до ~300 вакансий — это самый
затратный по токенам момент; дальше скорятся только новые.
